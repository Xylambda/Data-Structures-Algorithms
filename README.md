# Data-Structures-Algorithms
This repository provides implementations of popular algorithms and abstract data types using JAVA.

## Course
### Linear Abstract Data Types
1. List
   * ArrayList
   * LinkedList

2. Stack
   * ArrayStack
   * LinkedStack

3. Queue
   * ArrayQueue
   * LinkedQueue

### Set Abstract Data Types
1. Set
   * ArrayBit
   * LinkedSet

2. Priority Queue
   * Heap
   
3. Dictionary
   * Hash table

### Tree Abstract Data Types
1. Binary Search Tree
2. AVL Binary Search Tree

### Graph Abstract Data Types
1. Directed Graph
   * Adjacency List
   * Adjacency Matrix

2. Undirected Graph
   * Adjacency List
   * Adjacency Matrix
